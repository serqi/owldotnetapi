Return Value

An object that implements the IOwlLiteral interface. This is a reference to the newly added literal.

Implements

IOwlGraph.AddLiteral

Remarks

This method looks in the graph to determine whether a literal with the specified value, datatype and langID exists in the Graph. If the literal exists a reference to the existing literal is returned. If it does not exist then a new literal with the specified value, datatype and LangID is created, added to the graph and returned. Any parameter supplied to this method, except literalValue, can be null or empty and it will be ignored.

Exceptions

Exception Type Condition
UriFormatException The specified datatype URI is not null and is not a well formed URI.
