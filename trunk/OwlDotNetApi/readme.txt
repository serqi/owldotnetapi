OwlDotNetApi README
===================

Welcome to OwlDotNetApi. The OwlDotNetApi is an OWL (Web Ontology Language) 
API and parser written in C# for the .NET platform based on the Drive RDF 
parser. Its fully compliant with the W3C OWL syntax specification and can be 
used within any .NET language. 

Documentation is in the doc/ and can be found on the web at
http://mach.vub.ac.be/~bpellens/OwlDotNetApi/index.html
